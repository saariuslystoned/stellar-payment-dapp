# Get Public Address (G...)

Retrieves the public address for a Stellar account stored in the local keystore.

## Usage

// turbo
Run the following command, replacing `$ACCOUNT_NAME` with the account name (e.g., `buyer`, `seller`, `oracle`):

```bash
stellar keys address $ACCOUNT_NAME
```

## Available Accounts

Run `stellar keys ls` to see all available account names.
